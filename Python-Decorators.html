Python Decorators 浅析

发表于 2020-05-10 |

来源：https://realpython.com/primer-on-python-decorators/

A decorator is a function that takes another function and extends the behavior of the latter function without explicitly modifying it.
(装饰器是一个函数，它接收另一个函数，无需显式修改，而是可以扩展其函数功能)

First-Class Objects 第一类对象

In Python, functions are first-class objects. This means that functions can be passed around and used as arguments, just like any other object (string, int, float, list, and so on). Consider the following three functions:

say_hello() 和 be_awesome() 需要传入string。而greet_bob()函数期望一个函数作为参数传入，我们可以将say_hello()，be_awesome() 传入。

The say_hello function is named without parentheses(括号）. This means that only a reference to the function is passed. The function is not executed.
