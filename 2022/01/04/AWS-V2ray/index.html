AWS V2ray搭建

发表于 2022-01-04
使用AWS免费云（一年有效期） 一键搭建V2Ray，实现科学上网
1. 注册AWS云服务，需信用卡，启动EC2。 ![](/images/AWS_V2ray/1.png) 2. 选择带符合免费套餐 Ubuntu 18 ![](/images/AWS_V2ray/2.png) 3. 默认，下一步：配置实际详细信息 ![](/images/AWS_V2ray/3.png) 4. 默认，添加存储 ![](/images/AWS_V2ray/4.png) 5. 下一步，配置安全组 ![](/images/AWS_V2ray/5.png) 6. 选所有流量，任何位置 ![](/images/AWS_V2ray/6.png) 7. 审核和启动 ![](/images/AWS_V2ray/7.png) 8. AWS让你创建密钥，以便远程登录。生成证书yangxiukun.pem文件，启动成功 ![](/images/AWS_V2ray/8.png) ![](/images/AWS_V2ray/9.png) ![](/images/AWS_V2ray/10.png) 9. 查看实例已成功，服务器位于日本。复制IVP4公有IP 下面的IP地址 ![](/images/AWS_V2ray/11.png)


10. windows下载putty，打开PuttyGen工具。点击Load加载.pem证书。点击Save private key来保存.ppk证书。然后到SSH-Auth 选择刚刚生成的ppk证书。

11. Host Name 填上AWS的IP地址，port 22.

12. 进入服务器内的ubuntu系统。

13.新的一键V2Ray脚本
bash <(curl -s -L https://git.io/v2ray-setup.sh)

v2ray info 查看 V2Ray 配置信息
v2ray config 修改 V2Ray 配置
v2ray link 生成 V2Ray 配置文件链接
v2ray infolink 生成 V2Ray 配置信息链接
v2ray qr 生成 V2Ray 配置二维码链接
v2ray ss 修改 Shadowsocks 配置
v2ray ssinfo 查看 Shadowsocks 配置信息
v2ray ssqr 生成 Shadowsocks 配置二维码链接
v2ray status 查看 V2Ray 运行状态
v2ray start 启动 V2Ray
v2ray stop 停止 V2Ray
v2ray restart 重启 V2Ray
v2ray log 查看 V2Ray 运行日志
v2ray update 更新 V2Ray
v2ray update.sh 更新 V2Ray 管理脚本
v2ray uninstall 卸载 V2Ray

生成二维码链接，手机下载shadowrocket（美区appleid），扫码二维码。

14.打开youtube，测试连击情况
