Advantages of Properties in Python

Properties can be considered the “Pythonic” way of working with attributes because：

  • The syntax used to define properties is very concise(简洁) and readable.
  • You can access instance attributes exactly as if they were public attributes while using the “magic” of intermediaries (getters and setters) to validate(验证)new values and to avoid accessing or modifying the data directly(避免直接获取或修改数据）.
  • by using @property, you can “reuse” the name of a property to avoid creating new names for the getters, setters, and deleters.
