红黑树属于二叉搜索树，插入动作也与二叉搜索树一致，只不过红黑树插入后，多了平衡动作（旋转与涂色）
新插入的节点均为红色节点，因为红色不会影响路径上黑色节点的数量。

红黑树的插入平衡：

  1. N 为根节点 —-> N 涂黑
  2. N的父黑 —> 无需操作
  3. N的父红，叔红 —> p和u节点涂黑，GP涂红，再以GP为新的N，递归向上判断
  4. N的父红，叔黑。分情况
    4.1： N节点与父节点在同一边。
    4.1.1：N节点与父节点同在左边 -----> p涂黑，GP涂红，以GP右旋
    4.1.2：N节点与父节点同在右边 -----> p涂黑，GP涂红，以GP左旋
4.2： N节点与父节点P不同边。
    4.2.1：父节点在左，N节点在右边 ----> 以p左旋，执行4.1.1。（注意旋转后，P节点在最左边）
    4.2.2：父节点在右，N节点在左边 ----> 以p右旋，执行4.1.2。（注意旋转后，p节点在最右边）

希尔排序

发表于 2020-10-23 |

希尔排序： 一种分组插入的排序算法

基本思想： 已知数组长度为n,首先选取一个整数d1 = n//2 ，将元素分为d1个组，每组相邻量元素距离为d1，在各组内进行插入排序。 取第二个整数d2 = d1//2 重复上述分组排序过程，直到元素距离d为1，即所有元素在同一组内进行直接插入排序。

希尔排序每走一趟使数据整体越来越接近有序，最后一趟排序使所有数据有序。

堆的相关概念

二叉树：每个节点最多有两个子树的数结构，一个节点只能分两叉。

完全二叉树：二叉树的层数为h，除最后一层外，其他层都必须挂满，而且最后一层叶子节点得从左往右依次排列。

堆：是完全二叉树，分为大根堆和小根堆。大根堆是父节点比子节点大，小根堆是父节点比子节点小。

堆的存储：用数组来表示堆，若root节点存在序号0处，那么第i节点得父节点下标就为取整(i-1)/2。该节点得左右子节点为 2i+1，2i+2.
（如果root节点从1开始，那左右孩子节点分别为2i，2i+1.）

Divide and Conquer Algorithm(分而治之算法)

发表于 2020-08-25 |

Divide And Conquer
This technique can be divided into the following three parts:

  1. Divide: This involves dividing the problem into some sub problem.（将问题分成子问题）
  2. Conquer: Sub problem by calling recursively until sub problem solved.（通过递归调用解决子问题）
  3. Combine: The Sub problem Solved so that we will get find problem solution.

The following are some standard algorithms that follows Divide and Conquer algorithm.

  1. Binary Search is a searching algorithm. In each step, the algorithm compares the input element x with the value of the middle element in array. If the values match, return the index of the middle. Otherwise, if x is less than the middle element, then the algorithm recurs for left side of middle element, else recurs for the right side of the middle element.
  2. Quicksort is a sorting algorithm. The algorithm picks a pivot element, rearranges(重新排列) the array elements in such a way that all elements smaller than the picked pivot element move to left side of pivot, and all greater elements move to right side. Finally, the algorithm recursively sorts the subarrays on left and right of pivot element.
  3. Merge Sort is also a sorting algorithm. The algorithm divides the array in two halves, recursively sorts them and finally merges the two sorted halves.

树的定义和属性

树是一种将元素分层次存储的抽象数据类型。
最顶部元素为树根(root),除了最顶部的元素，每个元素在数中都有一个双亲节点和零个或者多个孩子节点。

通常我们将数定义为存储一系列元素的有限节点集合：

  • 如果数T不为空，则它一定具有一个称为 根节点 的特殊节点，并且该节点没有父节点。
  • 每个非根节点V都具有唯一的父节点w，每个具有父节点w的节点都是节点w的一个孩子。

有序树：如果数中每个节点的孩子节点都有特定的顺序，则称该树为有序树。我们将一个节点的孩子节点依次编号第一个，第二个，第三个等。通常我们按照从左到右的顺序对兄弟节点进行排序。

栈

栈是最简单的数据结构，先进后出。
抽象数据类型(ADT)：

  • S.push(e):将一个元素e添加到栈S的栈顶
  • S.pop():将栈顶元素移除并返回栈顶的元素，如果此时栈为空的，操作将出错。
  • S.top():不移除栈顶元素，返回一个栈S的栈顶元素；若为空，操作将出错。
  • S.is_empty() : 如果栈中不包含任何元素，返回布尔值“True”
  • len(s): 返回栈S中元素的数量。用 __ len __ 这个方法实现它。
