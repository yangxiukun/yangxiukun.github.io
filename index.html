python 基于数组的序列

2022-01-02

python中各种类，即内嵌的列表类(list)，元组类(tuple)和字符串类(str),这些类之间有明显的共性，最主要的是：每个类都支持用下标访问序列元素，每个类都使用数组这种低层次概念表示序列。计算机系统拥有庞大的储存字节，为了能跟踪信息储存在哪个字节，计算机采用了一个抽象概念，即存储地址。计算机内存，其每个字节都被指定了连续的存储地址。我们认为存储器的任一单个字节被存储或检索的运行时间为O(1).
一组相关变量能够一个接一个地存储在计算机存储器的一块连续区域内。我们将这样的表示法称为 数组（Array）
数组的每个单元必须占据相同数量的字节。之所以这样要求，是允许使用索引值能在常量单位内访问数组内的任一单元。

red_black_tree

2021-01-03

红黑树属于二叉搜索树，插入动作也与二叉搜索树一致，只不过红黑树插入后，多了平衡动作（旋转与涂色）
新插入的节点均为红色节点，因为红色不会影响路径上黑色节点的数量。

红黑树的插入平衡：

  1. N 为根节点 —-> N 涂黑
  2. N的父黑 —> 无需操作
  3. N的父红，叔红 —> p和u节点涂黑，GP涂红，再以GP为新的N，递归向上判断
  4. N的父红，叔黑。分情况
     4.1： N节点与父节点在同一边。
     4.1.1：N节点与父节点同在左边 -----> p涂黑，GP涂红，以GP右旋
     4.1.2：N节点与父节点同在右边 -----> p涂黑，GP涂红，以GP左旋
 4.2： N节点与父节点P不同边。
     4.2.1：父节点在左，N节点在右边 ----> 以p左旋，执行4.1.1。（注意旋转后，P节点在最左边）
     4.2.2：父节点在右，N节点在左边 ----> 以p右旋，执行4.1.2。（注意旋转后，p节点在最右边）

递归

递归是一种技术，这种技术通过一个函数在执行过程中一次或多次调用其本身
递归函数的优点是定义简单，逻辑清晰。理论上，所有的递归函数都可以写成循环的方式，但循环的逻辑不如递归清晰。
使用递归函数需要注意防止栈溢出。在计算机中，函数调用是通过栈（stack）这种数据结构实现的，每当进入一个函数调用，栈就会加一层栈帧，每当函数返回，栈就会减一层栈帧。由于栈的大小不是无限的，所以，递归调用的次数过多，会导致栈溢出
