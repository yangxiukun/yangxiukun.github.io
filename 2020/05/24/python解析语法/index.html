Comprehensions in Python provide us with a short and concise way to construct new sequences (such as lists, set, dictionary etc.) using sequences which have been already defined. (一种简短且简洁的方法，使用已经定义的序列来构造新序列)
Python supports the following 4 types of comprehensions:

  1. List Comprehensions: [ ]
  2. Dictionary Comprehensions:{ key:value }
  3. Set Comprehensions :{ }
  4. Generator Comprehensions :( )

list comprehensions

List comprehensions ：It consists of square brackets containing an expression followed by a for clause, then zero or more for or if clauses. The expressions can be anything, meaning you can put in all kinds of objects in lists. The result would be a new list made after the evaluation of the expression in context of the if and for clauses.

output_list = [output_exp for var in input_list if (var satisfies this condition)]
